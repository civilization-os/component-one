# Runtime tradeoff matrix

## Runtime tradeoff matrix

This slide demonstrates the dedicated quadrant analysis block for planning and strategy decks.

::: quadrant {#runtime-tradeoff-matrix}
x-axis: Delivery confidence
y-axis: Presentation ambition
top-left: Stable core | Double down on deterministic rendering, validation, and playback QA.
top-right: Rich stage language | Add stronger motion and guided emphasis while preserving predictable control.
bottom-left: Superficial polish | Avoid spending the cycle on editor chrome before runtime depth is stable.
bottom-right: Heavy choreography | Push media sequencing and complex scene logic until the model boundary settles.
:::

The point is to give analysis-heavy presentations a first-class visual primitive instead of forcing a table.
